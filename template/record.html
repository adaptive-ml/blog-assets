Interactive Visualizations with Shallot

This template demonstrates shallot-powered blog visualizations.

Demo 1: Step-Based Animation

Interactive sequences triggered by navigation buttons. Use the arrows or keyboard to step through animations.

Demo 2: Passive Animation

Continuous animation with shapes following a helix pattern. Runs automatically with no user interaction.

Demo 3: Grid Arrangement

Interactive grid layout with controllable arrangement. Use the UI controls to adjust grid parameters.

See blog/src/ for plugins and public/scenes/ for scene definitions.
